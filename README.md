# Slab fMRI Preprocessing Pipeline

* Optimized for partial-brain slab coverage
* generates basic quality control metrics
* uses TOPUP for SDC
* supports complex-valued BOLD data

Implemented in nipype  
Depends on ANTs, AFNI and FSL
